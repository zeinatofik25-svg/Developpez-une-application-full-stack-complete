## FAQ utilisateur

### Inscription / Connexion

Q : Quel format de mot de passe est accepté ?
Minimum 6 caractères avec au moins une lettre, un chiffre et un caractère spécial. Exemple : `Mdd2026!`

Q : Email ou pseudo déjà utilisé à l'inscription.
Un compte existe déjà avec cet email ou ce pseudo. Changez l'un ou l'autre, ou connectez-vous directement.

Q : Identifiants invalides à la connexion.
Vérifiez votre email/pseudo et mot de passe. Le champ `identifier` accepte les deux.

Q : Erreur 401 alors que je suis connecté.
Le token JWT expire après 24 h. Reconnectez-vous.

### Sujets et feed

Q : Mon feed est vide.
Abonnez-vous à au moins un sujet dans la page de Thèmes.

Q : Je ne trouve pas un sujet dans le formulaire de création d'article.
Seuls vos sujets abonnés apparaissent dans ce formulaire.

Q : Comment s’abonner à un sujet ?
Accédez à la page "Thèmes". Une liste de sujets liés à la programmation s’affiche. Cliquez sur le bouton "S’abonner" à côté du thème souhaité. Le bouton devient alors "Déjà abonné".

Q : Comment se désabonner d’un sujet ?
Rendez‑vous sur la page "Profil". Dans la section abonnements, vous pouvez vous désabonner d’un sujet en cliquant sur l’option correspondante.

### Articles et commentaires

Q : Le bouton Envoyer du commentaire est gris.
Le textarea ne doit pas etre vide ou contenir uniquement des espaces.

Q : Comment consulter un article et ses commentaires ?
Cliquez sur un article dans le fil d’actualité. La page détaillée de l’article affichera le contenu complet ainsi que les commentaires des autres utilisateurs.

Q : Comment ajouter un commentaire à un article ?
Dans la page de l’article, écrivez votre message dans le champ de commentaire puis validez. Le commentaire sera ajouté immédiatement sous l’article.

### Profil

Q : Comment modifier mes informations de profil ?
Accédez à la page "Profil". Vous pourrez modifier votre e‑mail, votre nom d’utilisateur ou votre mot de passe, puis enregistrer les modifications.
 
Q : Où trouver tous les thèmes sur lesquelles je suis abonné?
Accéder à la page "Profil" pour trouver tous vos abonnements.
